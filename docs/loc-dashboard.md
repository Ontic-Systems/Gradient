## What this measures, what it does not

This dashboard is a **rough trust artifact**, not a verified bootstrap claim. It tells you:

- ✅ The relative size of the Rust kernel vs the self-hosted `.gr` tree.
- ✅ How much Gradient code lives in the repo as stdlib + trust corpus (which the compiler must accept on every CI green).

It does **not** tell you:

- ❌ Whether the self-hosted compiler actually executes (it does not yet — see [docs/SELF_HOSTING.md](SELF_HOSTING.md)).
- ❌ Which kernel surfaces are `SelfHostedDefault` vs `Hybrid` (see `codebase/compiler/src/kernel_boundary.rs`).
- ❌ Whether the Rust kernel is \"minimal\" in the post-#116 sense.

For the honest split between Rust-hosted alpha, self-hosting bootstrap alpha, and true self-hosted compiler alpha, see [`docs/SELF_HOSTING.md`](SELF_HOSTING.md) and the \"alpha-readiness assessment\" section of the internal handoff.
